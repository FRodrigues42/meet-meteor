meteor.com

  • Deploy to meteor servers yourapp.meteor.com 
    meteor deploy yourapp
  • Convenient
  • Free
  • Custom domain
  • --password
  • Unused for an hour - process terminated (or whenever it suits meteor.com)

Other hosted

Other known to work hosted options:

  • Heroku
  • Dotcloud
  • Amazon EC2

DIY

VM's are super cheap nowadays. Install Meteor.

  • Run it in production mode: 
    
                meteor --production
    (Minify and bundle CSS and JS files)
  • or: 
    
                meteor bundle
    ...then extract and run as node
