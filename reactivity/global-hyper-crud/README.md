Reactivity! - Coping with change 
================================

In your terminal, run `meteor` in this directory:
	
	meteor
	open http://localhost:3000

Now you've got a simple Meteor app running. If you can see:

	=> Meteor server running on: http://localhost:3000/

...then all is well.
